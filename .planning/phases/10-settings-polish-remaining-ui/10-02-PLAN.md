Standard CATCH (canonical UNAUTHORIZED short-circuit from ProgressScreen.tsx + dictionary mapping):
```tsx
} catch (err) {
  if (err instanceof Error && err.message === 'UNAUTHORIZED') {
    router.replace(`/${locale}/login`)
    return
  }
  // For ApiRequestError: map by status to friendly dictionary message
  if (err instanceof ApiRequestError) {
    if (err.status >= 500) {
      setError(dict.errors.generic)
    } else if (err.status === 0 /* network */) {
      setError(dict.errors.offline)
    } else {
      // 400-level: use scoped dictionary key if one exists, else generic
      setError(dict.<scope>.err<Action>Failed ?? dict.errors.generic)
    }
    return
  }
  setError(dict.errors.generic)
}
```

**IMPORTANT — Canonical UNAUTHORIZED pattern:** The check is `err.message === 'UNAUTHORIZED'` (matches ProgressScreen.tsx line 45). Do NOT use `err.status === 401` for the redirect check — UNAUTHORIZED is thrown as a plain `Error` by upstream helpers (e.g., `lib/progress.ts`), not as an `ApiRequestError`. Using status-based 401 would miss the canonical case.
